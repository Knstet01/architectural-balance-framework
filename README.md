## О чём это

Гипотеза предела описаний утверждает: любая система, где накапливаются описания, растёт плотность связей и исчезает внешний взгляд, неизбежно достигает грани, за которой описание начинает замещать живое.

**Центральный документ:** [Таблица предела описаний](./Закон%20предела%20описаний.md)

---

## Формула

**Отклонение = (сцепление / предел сцепления) × (поток / предел потока) × (память / предел памяти) × (1 / доля взгляда со стороны)**

---

## Три порога

| Отклонение | Режим |
|------------|-------|
| **< 1** | Знание служит человеку |
| **1–2** | Человек служит знанию |
| **> 2** | Знание служит знанию. Человек вычеркнут |
